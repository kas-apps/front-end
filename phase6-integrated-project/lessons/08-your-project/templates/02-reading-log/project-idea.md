# プロジェクトアイデア：読書記録アプリ 📚

---

## プロジェクト概要

### 何を作るか

読んだ本を記録し、読書目標を管理できるWebアプリケーション

### 誰が使うか

- 本を読むのが好きな人
- 読書習慣を継続したい人
- 読んだ本を記録したい人

### どんな問題を解決するか

- 読んだ本のタイトルを忘れてしまう
- 「あの本、面白かったな」と思っても内容を思い出せない
- 読書目標を立てても、進捗が分からない
- 読みたい本リストを管理したい

```text

なぜなら「今年は50冊読む」という目標を達成したいから
```

## 機能要件の例

### 必須機能（Must Have）

#### F-001：本の追加

- タイトル（必須、最大200文字）
- 著者（任意、最大100文字）
- 評価（必須、1-5の星評価）
- 感想（任意、最大1000文字）
- 読了日（任意、日付）
- ジャンル（必須、小説・ビジネス・技術書・その他）
- ステータス（必須、読みたい・読んでいる・読了）

#### F-002：本の一覧表示

- カード形式で表示
- タイトル、著者、評価、ジャンル、ステータスを表示
- レスポンシブデザイン

#### F-003：本の削除

- 削除ボタンをクリック
- 確認ダイアログ表示

#### F-004：読書統計

- 全登録冊数
- ステータス別の冊数（読みたい・読んでいる・読了）
- 今月読んだ冊数

### 拡張機能（Should Have）

#### F-101：本の編集

#### F-102：ステータスフィルター

#### F-103：ジャンルフィルター

#### F-104：評価順でソート

#### F-105：タイトル・著者で検索

#### F-106：データの永続化

## 実装のヒント

### 星評価の表示

```javascript
function renderStars(rating) {
  const filledStars = '⭐'.repeat(rating);
  const emptyStars = '☆'.repeat(5 - rating);
  return filledStars + emptyStars;
}

// 使用例
renderStars(4); // "⭐⭐⭐⭐☆"
```

### ステータスバッジの色分け

```javascript
function getStatusBadgeClass(status) {
  const classes = {
    want: 'bg-gray-500',
    reading: 'bg-blue-500',
    finished: 'bg-green-500'
  };
  return classes[status];
}
```

### 今月読んだ冊数の計算

```javascript
function countBooksThisMonth() {
  const now = new Date();
  const thisMonth = `${now.getFullYear()}-${String(now.getMonth() + 1).padStart(2, '0')}`;

  return books.filter(book =>
    book.status === 'finished' &&
    book.finishedDate &&
    book.finishedDate.startsWith(thisMonth)
  ).length;
}
```
