# プロジェクトアイデア：支出管理アプリ 💰

---

## プロジェクト概要

### 何を作るか

日々の支出を記録し、カテゴリ別に集計できるWebアプリケーション

### 誰が使うか

- 家計を管理したい人
- 支出を記録したい人
- お金の使い方を見直したい人

### どんな問題を解決するか

- 「今月いくら使ったっけ？」が分からない
- 何にお金を使っているか把握できない
- カテゴリ別に支出を集計したい
- 予算を決めても、守れているか分からない

## 機能要件の例

### 必須機能（Must Have）

#### F-001：支出の追加

- 金額（必須、数値）
- カテゴリ（必須、食費・交通費・娯楽・光熱費・その他）
- 日付（必須、デフォルト：今日）
- メモ（任意、最大200文字）

#### F-002：支出の一覧表示

- カード形式またはリスト形式で表示
- 金額、カテゴリ、日付、メモを表示

#### F-003：支出の削除

- 削除ボタンをクリック
- 確認ダイアログ表示

#### F-004：支出統計

- 今月の合計支出
- カテゴリ別の支出額
- 今週の支出

### 拡張機能（Should Have）

#### F-101：支出の編集

#### F-102：カテゴリフィルター

#### F-103：日付範囲でフィルター

#### F-104：金額でソート

#### F-105：メモで検索

#### F-106：月間予算の設定と進捗表示

## 実装のヒント

### 金額のフォーマット

```javascript
function formatAmount(amount) {
  return `¥${amount.toLocaleString('ja-JP')}`;
}

// 使用例
formatAmount(1200); // "¥1,200"
formatAmount(45000); // "¥45,000"
```

### カテゴリアイコンの設定

```javascript
function getCategoryIcon(category) {
  const icons = {
    food: '🍔',
    transport: '🚇',
    entertainment: '🎬',
    utilities: '💡',
    other: '📦'
  };
  return icons[category] || '📦';
}
```

### カテゴリ別の合計計算

```javascript
function calculateCategoryTotals() {
  const totals = {
    food: 0,
    transport: 0,
    entertainment: 0,
    utilities: 0,
    other: 0
  };

  expenses.forEach(expense => {
    totals[expense.category] += expense.amount;
  });

  return totals;
}
```

### 今月の支出フィルター

```javascript
function getThisMonthExpenses() {
  const now = new Date();
  const thisMonth = `${now.getFullYear()}-${String(now.getMonth() + 1).padStart(2, '0')}`;

  return expenses.filter(expense =>
    expense.date.startsWith(thisMonth)
  );
}
```
