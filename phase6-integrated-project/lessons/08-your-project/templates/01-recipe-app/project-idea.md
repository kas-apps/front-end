# プロジェクトアイデア：レシピ管理アプリ 👨‍🍳

---

## プロジェクト概要

### 何を作るか

お気に入りのレシピを保存・管理できるWebアプリケーション

### 誰が使うか

- 料理が好きな人
- レシピをたくさん試す人
- レシピを整理したい人

### どんな問題を解決するか

- レシピをメモ帳に書いているが、見つけにくい
- 「あのレシピ、どこに書いたっけ？」と探すのに時間がかかる
- ジャンル別に整理したいが、紙だと難しい
- 調理時間や難易度で絞り込みたい

## 機能要件の例

### 必須機能（Must Have）

#### F-001：レシピの追加

- レシピタイトル（必須、最大100文字）
- 材料リスト（任意、テキストエリア）
- 手順（任意、テキストエリア）
- 調理時間（任意、数値＋単位）
- 難易度（必須、簡単・普通・難しい）
- ジャンル（必須、和食・洋食・中華・その他）

#### F-002：レシピの一覧表示

- カード形式で表示
- レシピタイトル、ジャンル、調理時間、難易度を表示
- モバイル1カラム、タブレット2カラム、デスクトップ3カラム

#### F-003：レシピの削除

- 削除ボタンをクリック
- 確認ダイアログ表示
- 削除実行

#### F-004：統計表示

- 全レシピ数
- ジャンル別のレシピ数

### 拡張機能（Should Have）

#### F-101：レシピの編集

- 編集ボタンをクリックでインライン編集モード
- 保存・キャンセルボタン

#### F-102：フィルター機能

- ジャンルフィルター（すべて・和食・洋食・中華・その他）
- 難易度フィルター（すべて・簡単・普通・難しい）

#### F-103：ソート機能

- 作成日順（新しい順・古い順）
- 調理時間順（短い順・長い順）
- タイトル順（昇順・降順）

#### F-104：検索機能

- レシピタイトルと材料から検索

#### F-105：データの永続化

- LocalStorageに保存

## TODOアプリとの違い

### 類似点

- カード形式の一覧表示
- 追加・編集・削除機能
- フィルター・ソート・検索機能
- LocalStorageでの永続化

### 相違点

| 項目 | TODOアプリ | レシピ管理アプリ |
|------|-----------|----------------|
| メインデータ | タスク | レシピ |
| 複数行テキスト | 説明のみ | 材料、手順 |
| カテゴリ | 仕事・プライベート・買い物 | 和食・洋食・中華・その他 |
| ステータス | 未完了・完了 | なし |
| 数値データ | なし | 調理時間 |

### 応用のポイント

```javascript
// TODOアプリのタスクオブジェクト
{
  id: number,
  title: string,
  description: string,  // ← 1つのテキストエリア
  priority: string,
  category: string,
  completed: boolean    // ← 完了フラグ
}

// レシピアプリのレシピオブジェクト
{
  id: number,
  title: string,
  ingredients: string,   // ← 材料リスト（複数行）
  instructions: string,  // ← 手順（複数行）
  cookingTime: number,   // ← 数値データ
  difficulty: string,
  genre: string          // ← ジャンル（カテゴリ）
  // completed フラグはなし
}
```

## 実装のヒント

### 1. 複数行テキストの表示

```javascript
// 材料リストを改行で区切って表示
function renderIngredients(ingredients) {
  return ingredients.split('\n')
    .map(item => `<li>${item}</li>`)
    .join('');
}

// HTML
<ul class="list-disc list-inside">
  ${renderIngredients(recipe.ingredients)}
</ul>
```

### 2. 調理時間の表示

```javascript
function formatCookingTime(minutes) {
  if (!minutes) return 'なし';
  if (minutes < 60) return `${minutes}分`;

  const hours = Math.floor(minutes / 60);
  const mins = minutes % 60;
  return mins > 0 ? `${hours}時間${mins}分` : `${hours}時間`;
}
```

### 3. 難易度バッジの色分け

```javascript
function getDifficultyBadgeClass(difficulty) {
  const classes = {
    easy: 'bg-green-500',
    medium: 'bg-yellow-500',
    hard: 'bg-red-500'
  };
  return classes[difficulty] || 'bg-gray-500';
}

// HTML
<span class="${getDifficultyBadgeClass(recipe.difficulty)} text-white px-2 py-1 rounded">
  ${recipe.difficulty === 'easy' ? '簡単' : recipe.difficulty === 'medium' ? '普通' : '難しい'}
</span>
```
