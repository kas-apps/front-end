# Phase 1: HTML/CSS 基礎編 🎨

**目標**：静的なレイアウトでドキュメントを作成できるようになる（Word/Pages と同等レベル）

---

## この Phase で学ぶこと

Phase 0 で「Word/Pages と HTML/CSS は本質的に同じ」って話したの、覚えてる？

Phase 1 では、その知識を活かして、**実際に HTML/CSS で文書を作る**方法を学んでいくよ！

### できるようになること

- 見出しや段落を使った構造的な文書が作れる
- リストや表を使って情報を整理できる
- 画像やリンクを含む文書が作れる
- CSS で文書を美しくスタイリングできる
- セマンティックな HTML で意味のある文書構造を作れる

**これだけできれば、ブログ記事や会社の紹介ページくらいは余裕で作れるよ！** 💪

## 🚀 Phase 1 の進め方

### 推奨ステップ

1. **Lesson 1 から順番に進める**

   - 各レッスンは前のレッスンの知識を前提にしているよ
   - 焦らず、一つずつ確実に理解していこう

2. **実際に手を動かす**

   - サンプルコードを自分で書いてみる
   - AI に指示して、生成されたコードを観察する
   - 「なぜこうなるのか」を考えながら進める

3. **演習問題に挑戦する**

   - 各レッスンの演習問題を必ず解く
   - わからなければ AI に相談しながらで OK
   - 解答例と自分のコードを比較する

4. **ミニプロジェクトで実践**
   - Phase 1 の最後に、学んだことを全部使ったプロジェクトに挑戦
   - 自己紹介ページや簡単なブログ記事を作ってみよう

## 🤖 バイブコーダー向けポイント

### Phase 1 での AI の使い方

Phase 1 では、こんな風に AI を活用しよう：

#### 1. 基本的な構造を作ってもらう

```text
良い指示の例：
「HTML5の基本構造で、タイトルが『私の自己紹介』の
空のHTMLファイルを作ってください」

AIが生成したら、各部分の意味を理解しよう
```

#### 2. 要素の使い方を学ぶ

```text
良い指示の例：
「h1見出しと3つの段落を含む記事のHTMLを書いてください。
テーマは『私の趣味』でお願いします」

生成されたコードで、どの要素がどんな役割か確認しよう
```

#### 3. CSS でスタイリングする

```text
良い指示の例：
「見出しを青色、font-sizeを28pxにするCSSを書いてください」

生成されたCSSがどこに効いているか、ブラウザで確認しよう
```

### チェックポイント

AI が生成したコードを見る時、こんなポイントをチェックしよう：

- ✅ **HTML 構造が正しいか**

  - 開始タグと終了タグが対応している
  - 適切にネストされている
  - セマンティックな要素が使われている

- ✅ **CSS セレクタが適切か**

  - 意図した要素にスタイルが当たっている
  - クラス名が分かりやすい
  - ID とクラスを適切に使い分けている

- ✅ **可読性が高いか**
  - インデントが適切
  - コメントが必要な箇所にある
  - 命名が分かりやすい
