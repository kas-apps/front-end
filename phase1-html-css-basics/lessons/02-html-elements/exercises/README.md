## 基礎編

### 演習 1：自己紹介ページを作ろう

以下の要素を含む自己紹介ページを作ってください：

- h1：あなたの名前
- h2：「自己紹介」
- p：2 つの段落で自己紹介
- h2：「好きなもの」
- ul：好きなものを 3 つ

**解答例**：[02-01.html](solutions/02-01.html)

<details>
<summary>ヒントを見る</summary>

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>自己紹介</title>
  </head>
  <body>
    <h1><!-- あなたの名前 --></h1>

    <h2>自己紹介</h2>
    <p><!-- 自己紹介文1 --></p>
    <p><!-- 自己紹介文2 --></p>

    <h2>好きなもの</h2>
    <ul>
      <li><!-- 好きなもの1 --></li>
      <!-- 残りを追加 -->
    </ul>
  </body>
</html>
```

</details>

### 演習 2：レシピページを作ろう

料理のレシピページを作ってください：

- h1：料理名
- h2：「材料」
- ul：材料リスト（3 つ以上）
- h2：「作り方」
- ol：手順（3 ステップ以上）

**解答例**：[02-02.html](solutions/02-02.html)

---

### 演習 3：リンク集を作ろう

お気に入りの Web サイトのリンク集を作ってください：

- h1：「お気に入りサイト」
- 各サイトへのリンクを 3 つ
- 各リンクの下に、そのサイトの説明（p）

**解答例**：[02-03.html](solutions/02-03.html)

## 応用編

### 演習 4：商品一覧ページを作ろう

以下の要素を含むページを作ってください：

- h1：「商品一覧」
- 3 つの商品について：
  - h2：商品名
  - img：商品画像
  - p：商品説明
  - リンク：「詳細を見る」

**解答例**：[02-04.html](solutions/02-04.html)

### 演習 5：時間割を作ろう

学校や仕事の時間割をテーブルで作ってください：

- table で時間割を作成
- thead：曜日（月〜金）
- tbody：各時間帯のスケジュール
- 最低 5 行 ×5 列

**解答例**：[02-05.html](solutions/02-05.html)

## チャレンジ

### 演習 6：AI と一緒に作ろう

AI に以下を指示して、ブログ記事風のページを作ってもらおう：

1. 適切な指示を考えて、AI に伝える
2. 生成されたコードをチェックリストで確認
3. 問題があれば修正を指示
4. 完成したコードをブラウザで表示して確認

**含める要素**：

- 記事タイトル（h1）
- 複数のセクション（h2）
- 画像 2 枚以上
- リスト 1 つ以上
- 外部サイトへのリンク 1 つ以上

**AI への指示例**：

```text
「フロントエンド開発の魅力というテーマでブログ記事を書いてください。
以下の要素を含めてください：
- h1でメインタイトル
- h2で複数のセクション（イントロ、魅力、学習方法、おすすめリソース、まとめ）
- 画像を2枚（frontend-development.jpgとlearning-code.jpg）
- 魅力のリストをul、学習ステップをolで
- MDN Web Docsへの外部リンク
- meta descriptionも追加してください」
```

**解答例**：[02-06.html](solutions/02-06.html)
