簡単カレーライス

材料

  • カレールー：1箱（8皿分）
  • 豚肉（または鶏肉）：300g
  • 玉ねぎ：2個
  • にんじん：1本
  • じゃがいも：3個
  • 水：800ml
  • サラダ油：大さじ2

作り方

  1. 野菜と肉を一口大に切る。玉ねぎはくし切り、にんじんとじゃがいもは乱切りにする。
  2. 鍋に油を熱し、肉を炒める。肉の色が変わったら、野菜を加えてさらに炒める。
  3. 水を加えて沸騰させ、アクを取りながら約15分煮込む。
  4. 一度火を止めてカレールーを割り入れ、よく溶かす。
  5. 再び弱火で約10分煮込み、とろみがついたら完成。ご飯と一緒に盛り付ける。
