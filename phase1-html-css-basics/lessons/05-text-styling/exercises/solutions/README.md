#### [05-01.html](05-01.html) - 演習 1 の解答：色を変更しよう

**演習内容**：

- h1 を青色（`#3b82f6`）にする
- p を灰色（`#6b7280`）にする

**ポイント**：

- ✅ `color` プロパティで文字色を指定
- ✅ 16 進数カラーコード（`#3b82f6`）を使用
- ✅ 要素セレクタ（`h1`, `p`）で適用

**16 進数カラーコードの読み方**：

```
#3b82f6
 ↓  ↓  ↓
 赤 緑 青
 3b 82 f6
```

- 各色を 00〜ff（0〜255）で表現
- 6 桁で色を指定（省略形：`#fff` = `#ffffff`）

#### [05-02.html](05-02.html) - 演習 2 の解答：フォントサイズを調整しよう（rem単位）

**演習内容**：

- h1 を `2.5rem`（40px）にする
- h2 を `2rem`（32px）にする
- p を `1rem`（16px）にする

**ポイント**：

- ✅ `html { font-size: 16px; }` でルート要素のサイズを設定
- ✅ `rem` 単位はルート要素（html）のフォントサイズを基準にする
- ✅ `2.5rem` = `2.5 × 16px` = `40px`

**rem 単位の計算**：

| 指定       | 計算式         | 結果  |
| ---------- | -------------- | ----- |
| `1rem`     | `1 × 16px`     | 16px  |
| `2rem`     | `2 × 16px`     | 32px  |
| `2.5rem`   | `2.5 × 16px`   | 40px  |
| `0.875rem` | `0.875 × 16px` | 14px  |

**rem のメリット**：

- ユーザーがブラウザのフォントサイズを変更した時に、すべての要素が比例して拡大/縮小される
- アクセシビリティに優れている（視覚障害のある人にも優しい）
- 一貫性のあるサイズ設定ができる

#### [05-03.html](05-03.html) - 演習 3 の解答：行間を調整しよう

**演習内容**：

- p の行間を `1.6` にする

**ポイント**：

- ✅ `line-height` プロパティで行間を指定
- ✅ 単位なしで指定すると、フォントサイズに対する倍率になる
- ✅ `line-height: 1.6;` = フォントサイズの 1.6 倍の行間

**line-height の指定方法**：

| 指定方法              | 意味                                           | 推奨度 |
| --------------------- | ---------------------------------------------- | ------ |
| `line-height: 1.6;`   | フォントサイズの 1.6 倍（単位なし）            | ⭐⭐⭐ |
| `line-height: 160%;`  | フォントサイズの 160%（親要素を継承する問題）  | ⭐⭐   |
| `line-height: 1.6em;` | フォントサイズの 1.6 倍（親要素を継承する問題） | ⭐⭐   |
| `line-height: 24px;`  | 固定値 24px（フォントサイズ変更時に対応しない） | ⭐     |

**実務では単位なし（`1.6`）が最もおすすめ！**

**読みやすい行間の目安**：

- 本文：`1.5` 〜 `1.8`（一般的に `1.6` が読みやすい）
- 見出し：`1.2` 〜 `1.4`（小さめが引き締まる）
- コード：`1.4` 〜 `1.6`（少し詰める）

## 応用編

#### [05-04.html](05-04.html) - 演習 4 の解答：見出しをスタイリングしよう

**演習内容**：

- h1 の文字色を青（`#3b82f6`）にする
- h1 のフォントサイズを `2.5rem` にする
- h1 を中央寄せにする
- h1 の文字間隔を `0.05em` にする
- h1 を太字にする

**ポイント**：

- ✅ `color` で文字色を指定
- ✅ `font-size` でサイズを指定
- ✅ `text-align: center` で中央寄せ
- ✅ `letter-spacing` で文字間隔を指定
- ✅ `font-weight: bold` で太字に

**text-align の値**：

| 値        | 意味       | 用途                     |
| --------- | ---------- | ------------------------ |
| `left`    | 左寄せ     | 本文（デフォルト）       |
| `center`  | 中央寄せ   | 見出し、キャッチコピー   |
| `right`   | 右寄せ     | 日付、署名               |
| `justify` | 両端揃え   | 新聞のような文章（日本語では注意） |

**letter-spacing の単位**：

- `em`：フォントサイズに対する相対値（推奨）
- `px`：絶対値（固定サイズ）
- `0.05em` = フォントサイズの 0.05 倍の間隔

**font-weight の値**：

| 値       | 太さ   | よく使う値 |
| -------- | ------ | ---------- |
| `normal` | 標準   | `400`      |
| `bold`   | 太字   | `700`      |
| `100`    | 極細   | -          |
| `900`    | 極太   | -          |

#### [05-05.html](05-05.html) - 演習 5 の解答：リンクをスタイリングしよう

**演習内容**：

- リンク（a）の文字色を青（`#3b82f6`）にする
- リンクの下線を消す
- ホバー時に下線を表示する

**ポイント**：

- ✅ `text-decoration: none` で下線を消す
- ✅ `:hover` 疑似クラスでホバー時のスタイルを指定
- ✅ `text-decoration: underline` で下線を表示

**疑似クラスとは**：

要素の特定の状態に対してスタイルを適用できる機能。

**リンクの疑似クラス**：

| 疑似クラス   | 意味                         | 例                      |
| ------------ | ---------------------------- | ----------------------- |
| `:link`      | 未訪問のリンク               | `a:link { ... }`        |
| `:visited`   | 訪問済みのリンク             | `a:visited { ... }`     |
| `:hover`     | マウスオーバー時             | `a:hover { ... }`       |
| `:active`    | クリック中                   | `a:active { ... }`      |
| `:focus`     | キーボードでフォーカス時（Tab キーなど） | `a:focus { ... }`       |

**疑似クラスの順序**（覚え方：**LoVe HAte**）：

```css
a:link { ... }
a:visited { ... }
a:hover { ... }
a:active { ... }
```

**text-decoration の値**：

| 値             | 意味             | 用途                     |
| -------------- | ---------------- | ------------------------ |
| `none`         | 装飾なし         | リンクの下線を消す       |
| `underline`    | 下線             | リンクの下線を表示       |
| `overline`     | 上線             | あまり使わない           |
| `line-through` | 打ち消し線       | 削除された価格など       |

#### [05-06.html](05-06.html) + [05-06.css](05-06.css) - 演習 6 の解答：AIで完全なスタイリング

**演習内容**：

- AI に指示して包括的な CSS を生成
- 外部 CSS ファイルを作成

**ポイント**：

- ✅ 具体的な指示を出す（色、サイズ、配置を明示）
- ✅ 外部 CSS ファイル（`.css`）に分離
- ✅ `<link rel="stylesheet" href="05-06.css">` で読み込み
- ✅ 複数のスタイルを組み合わせる

**AI への指示のコツ**：

| 悪い指示                 | 良い指示                                                  |
| ------------------------ | --------------------------------------------------------- |
| 「見出しを青くして」     | 「h1を青色（#3b82f6）、40px、中央寄せにして」             |
| 「いい感じにして」       | 「bodyは薄い灰色背景、pは行間1.6、リンクはホバー時に下線」 |
| 「おしゃれにして」       | 「文字間隔0.05em、太字、内側余白10px」                    |

**生成された CSS のチェックポイント**：

1. **色の統一**：同じ青色（`#3b82f6`）が使われているか
2. **line-height は単位なし**：`line-height: 1.6;`（`1.6px` ではない）
3. **rem と px の使い分け**：フォントサイズは rem、余白は px が多い
4. **疑似クラスの記法**：`a:hover { ... }`（スペースなし）
5. **セミコロン**：各プロパティの最後に `;` があるか

**外部 CSS のメリット**：

- HTML と CSS を分離して管理しやすい
- 複数の HTML ファイルで同じ CSS を共有できる
- CSS の変更が全ページに反映される
- HTML が見やすくなる

## 🎯 学習のポイント

これらの解答例を見る時は、以下のポイントに注目しよう：

### 1. 色の指定方法

**16 進数カラーコードが実務では最も使われる！**

| 指定方法                      | 例                        | 用途                     |
| ----------------------------- | ------------------------- | ------------------------ |
| 色名                          | `color: blue;`            | 学習、プロトタイプ       |
| 16 進数                       | `color: #3b82f6;`         | 実務で最もよく使う       |
| RGB                           | `color: rgb(59, 130, 246);` | プログラムで生成する時   |
| RGBA（透明度あり）            | `color: rgba(59, 130, 246, 0.8);` | 半透明にしたい時         |
| HSL（色相・彩度・明度）       | `color: hsl(217, 91%, 60%);` | 色を調整しやすい         |

### 2. サイズ単位の使い分け

| 単位  | 基準               | 用途                           | 推奨度 |
| ----- | ------------------ | ------------------------------ | ------ |
| `px`  | 絶対値             | 固定サイズ、余白               | ⭐⭐   |
| `rem` | ルート要素（html） | フォントサイズ（実務でおすすめ！） | ⭐⭐⭐ |
| `em`  | 親要素             | 入れ子の要素（注意が必要）     | ⭐     |
| `%`   | 親要素             | レスポンシブデザイン           | ⭐⭐   |

**実務では `rem` をフォントサイズに使うのがスタンダード！**

### 3. テキスト関連プロパティ一覧

#### 色・サイズ

| プロパティ    | 意味           | 例                     |
| ------------- | -------------- | ---------------------- |
| `color`       | 文字色         | `color: #3b82f6;`      |
| `font-size`   | 文字サイズ     | `font-size: 2rem;`     |
| `font-family` | フォントの種類 | `font-family: Arial;`  |

#### 太さ・スタイル

| プロパティ    | 意味       | 例                      |
| ------------- | ---------- | ----------------------- |
| `font-weight` | 文字の太さ | `font-weight: bold;`    |
| `font-style`  | 文字スタイル | `font-style: italic;`   |

#### 装飾

| プロパティ         | 意味           | 例                            |
| ------------------ | -------------- | ----------------------------- |
| `text-decoration`  | 装飾線         | `text-decoration: underline;` |
| `text-transform`   | 大文字・小文字 | `text-transform: uppercase;`  |

#### 配置・間隔

| プロパティ        | 意味       | 例                         |
| ----------------- | ---------- | -------------------------- |
| `text-align`      | 水平配置   | `text-align: center;`      |
| `line-height`     | 行間       | `line-height: 1.6;`        |
| `letter-spacing`  | 文字間隔   | `letter-spacing: 0.05em;`  |
| `word-spacing`    | 単語間隔   | `word-spacing: 0.2em;`     |

### 4. よくある間違いとその修正

| 間違い                         | 問題点                       | 修正方法                      |
| ------------------------------ | ---------------------------- | ----------------------------- |
| `color #3b82f6;`               | コロンがない                 | `color: #3b82f6;`             |
| `color: #3b82f6`               | セミコロンがない             | `color: #3b82f6;`             |
| `line-height: 1.6px;`          | 単位なしで指定すべき         | `line-height: 1.6;`           |
| `a :hover { ... }`             | スペースがある               | `a:hover { ... }`             |
| `font-size: 2rem;` を em で計算 | rem はルート要素基準         | rem = html の font-size を基準 |
| `text-align: middle;`          | middle は存在しない          | `text-align: center;`         |

## 💡 自分で試してみよう

解答例を見たら、以下のことを試してみよう：

### 1. 色を変えてみる

- 青色を他の色に変える（`#ef4444` = 赤、`#10b981` = 緑）
- 背景色を追加する（`background-color: #f3f4f6;`）
- 透明度を調整する（`rgba(59, 130, 246, 0.5)`）

### 2. サイズを調整してみる

- フォントサイズを変える（`2.5rem` → `3rem`, `1.5rem` など）
- 行間を変えて読みやすさを比較する（`1.2`, `1.6`, `2.0`）
- 文字間隔を調整する（`0.05em`, `0.1em`, `0.2em`）

### 3. 新しいスタイルを追加

- テキストに影を付ける（`text-shadow: 2px 2px 4px gray;`）
- 太字や斜体を試す（`font-weight: bold;`, `font-style: italic;`）
- 大文字変換を試す（`text-transform: uppercase;`）
- フォントを変える（`font-family: Georgia, serif;`）

### 4. ブラウザで確認

- HTML ファイルをブラウザで開く
- 開発者ツール（F12）で CSS を確認・編集してみる
- リアルタイムでスタイルを変更して実験する

### 5. AI と協働

- 解答例を参考に、AI に別のスタイルを生成してもらう
- 「見出しにグラデーションを付けて」などの追加指示を出す
- 生成された CSS をチェックして、改善点を見つける
