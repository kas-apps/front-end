## 応用編

### 演習 4：外部 CSS を作ろう

演習 1 で作った内部 CSS を、外部 CSS ファイルに分離してください。

**手順**：

1. `04-04.css` という名前のファイルを作る
2. `<style>` タグの中身を CSS ファイルに移動する
3. HTML で `<link>` タグを使って CSS ファイルを読み込む

**解答例**：[04-04.html](solutions/04-04.html) + [04-04.css](solutions/04-04.css)

---

### 演習 5：AI に指示して CSS を生成しよう

AI に以下の指示を出して、CSS を生成してもらってください：

```text
「以下のHTMLに外部CSSを作成してください：
- h1は青色、36px、中央寄せ
- .card クラスは白背景、灰色のボーダー、内側20px余白、外側20px余白
- .highlight クラスは黄色背景、太字」
```

生成された CSS をチェックして、HTML ファイルから読み込めるようにしよう！

**解答例**：[04-05.html](solutions/04-05.html) + [04-05.css](solutions/04-05.css)
