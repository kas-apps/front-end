}

#### [04-03.html](04-03.html) - 演習 3 の解答：ボックスモデル

**演習内容**：
- カードのようなデザインを作成
- ボックスモデルの4つの要素を使用

**ポイント**：
- ✅ `background-color` で背景色を設定
- ✅ `border` で枠線を追加
- ✅ `padding` で内側の余白を設定（コンテンツと枠線の間）
- ✅ `margin` で外側の余白を設定（要素と要素の間）

**ボックスモデルの順序**：
```
margin（外側の余白）
  ↓
border（枠線）
  ↓
padding（内側の余白）
  ↓
content（コンテンツ）
```

## 応用編

#### [04-04.html](04-04.html) + [04-04.css](04-04.css) - 演習 4 の解答：外部CSS

**演習内容**：
- 内部CSSを外部CSSファイルに分離

**ポイント**：
- ✅ `<style>` タグの中身を `.css` ファイルに移動
- ✅ `<link rel="stylesheet" href="ファイル名.css">` で読み込み
- ✅ CSS ファイルには `<style>` タグは不要

**外部CSSのメリット**：
- 複数の HTML ファイルで同じ CSS を共有できる
- HTML と CSS を分離して管理しやすくなる
- CSS の変更が全ページに反映される

**よくある間違い**：
- ❌ CSS ファイルに `<style>` タグを書いてしまう
- ❌ `href` のパスが間違っている
- ❌ `<link>` タグを `<body>` に書いてしまう（`<head>` に書く！）

#### [04-05.html](04-05.html) + [04-05.css](04-05.css) - 演習 5 の解答：AIでCSS生成

**演習内容**：
- AI に指示して CSS を生成
- 外部 CSS ファイルを作成

**ポイント**：
- ✅ 具体的な指示を出す（色、サイズ、余白を明示）
- ✅ 生成された CSS をチェックする
- ✅ `text-align: center` で中央寄せを実現

**AI への指示のコツ**：
- プロパティ名を明示する（「青色」→「color: blue」）
- 数値を具体的に指定する（「大きめ」→「36px」）
- セレクタを明確に指定する（「見出し」→「h1」）

**生成された CSS のチェックポイント**：
1. セレクタが正しいか（`.class` vs `#id` vs `element`）
2. プロパティ名のスペルは正しいか
3. セミコロン（`;`）が各行の最後にあるか
4. 括弧（`{}`）が正しく閉じられているか

#### [04-06.html](04-06.html) - 演習 6 の解答：displayを使ってボタンを横に並べる

**演習内容**：
- `display: inline-block` でボタンを横並びにする
- ボタンのスタイルを設定

**ポイント**：
- ✅ `display: inline-block` で横並び + 幅指定可能
- ✅ `padding` で上下左右の余白を個別指定（`10px 20px` = 上下10px、左右20px）
- ✅ `margin-right` で右側だけ余白を追加

**display プロパティの使い分け**：

| 値             | 特徴                           | 用途                   |
| -------------- | ------------------------------ | ---------------------- |
| `block`        | 縦並び、幅指定可能             | 見出し、段落、div      |
| `inline`       | 横並び、幅指定不可             | リンク、span           |
| `inline-block` | 横並び、幅指定可能（いいとこ取り！） | ボタン、カードの横並び |

**padding の書き方**：
```css
/* 全方向同じ */
padding: 20px;

/* 上下、左右 */
padding: 10px 20px;

/* 上、左右、下 */
padding: 10px 20px 30px;

/* 上、右、下、左（時計回り） */
padding: 10px 20px 30px 40px;
```

## 🎯 学習のポイント

これらの解答例を見る時は、以下のポイントに注目しよう：

### 1. CSS の基本構文

```css
セレクタ {
  プロパティ: 値;
  プロパティ: 値;
}
```

**必ず守ること**：
- プロパティと値の間に `:` （コロン）
- 各プロパティの最後に `;` （セミコロン）
- 括弧 `{}` で囲む

### 2. セレクタの使い分け

| セレクタ         | 記法            | 用途                           |
| ---------------- | --------------- | ------------------------------ |
| 要素セレクタ     | `h1 { ... }`    | すべての h1 要素に適用         |
| クラスセレクタ   | `.card { ... }` | class="card" の要素に適用（最頻出！） |
| ID セレクタ      | `#header { ... }` | id="header" の要素に適用（1つだけ） |

**実務では クラスセレクタ（`.class`）を最もよく使う！**

### 3. ボックスモデルの理解

```
┌─────────────────────────────────┐
│  margin（外側の余白）            │
│  ┌───────────────────────────┐  │
│  │ border（枠線）             │  │
│  │ ┌─────────────────────┐   │  │
│  │ │ padding（内側の余白）│   │  │
│  │ │ ┌─────────────────┐ │   │  │
│  │ │ │ content（内容） │ │   │  │
│  │ │ └─────────────────┘ │   │  │
│  │ └─────────────────────┘   │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

### 4. よくある間違いとその修正

| 間違い                        | 問題点                           | 修正方法                         |
| ----------------------------- | -------------------------------- | -------------------------------- |
| `color blue;`                 | コロンがない                     | `color: blue;`                   |
| `color: blue`                 | セミコロンがない                 | `color: blue;`                   |
| `.card`（括弧なし）           | 括弧がない                       | `.card { ... }`                  |
| `<style>` を CSS ファイルに書く | CSS ファイルには不要             | `<style>` タグを削除             |
| `class=highlight`（引用符なし） | HTML の属性値に引用符が必要      | `class="highlight"`              |
| `padding: 10px20px`           | スペースがない                   | `padding: 10px 20px;`            |

## 💡 自分で試してみよう

解答例を見たら、以下のことを試してみよう：

### 1. 値を変えてみる

- 色を変える（`blue` → `red`, `green` など）
- サイズを変える（`32px` → `48px`, `16px` など）
- 余白を変える（`20px` → `10px`, `30px` など）

### 2. 新しいスタイルを追加

- 背景色を追加（`background-color: lightblue;`）
- 角を丸くする（`border-radius: 10px;`）
- 影を付ける（`box-shadow: 2px 2px 5px gray;`）

### 3. ブラウザで確認

- HTML ファイルをブラウザで開く
- 開発者ツール（F12）で CSS を確認・編集してみる
- リアルタイムでスタイルを変更して実験する

### 4. AI と協働

- 解答例を参考に、AI に別のスタイルを生成してもらう
- 「カードに影を付けて、角を丸くして」などの追加指示を出す
- 生成された CSS をチェックして、改善点を見つける
