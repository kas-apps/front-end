## 基礎編

### 演習 1：最初の HTML ファイルを作ろう

以下の条件で HTML ファイルを作成してください：

- タイトル：「自己紹介」
- 言語：日本語
- body 内容：
  - 見出し（h1）：あなたの名前
  - 段落（p）：簡単な自己紹介文

**ヒント**：レッスンで学んだ完全版テンプレートをベースに、body 部分を変更しよう！

**解答例**：[01-01.html](solutions/01-01.html)

---

### 演習 2：説明文を追加しよう

演習 1 で作ったファイルに、`<meta name="description">`を追加してください。

内容は「〇〇の自己紹介ページです」のような説明文にしよう。

**解答例**：[01-02.html](solutions/01-02.html)

---

### 演習 3：AI に指示して作ってもらおう

AI に以下の内容の HTML ファイルを作ってもらってください：

- タイトル：「私の好きな食べ物」
- body 内容：
  - h1 見出し：「好きな食べ物トップ 3」
  - 3 つの段落、それぞれ好きな食べ物について

生成されたコードを、レッスンで学んだ「チェックリスト」で確認してみよう！

#### ✅ チェックリスト

1. **DOCTYPE 宣言があるか？**
   - `<!DOCTYPE html>` がファイルの先頭にあるか
2. **lang 属性が正しいか？**
   - `<html lang="ja">` となっているか
3. **文字コード指定があるか？**
   - `<meta charset="UTF-8">` があるか
4. **title タグがあるか？**
   - `<title>` タグに適切なタイトルが設定されているか
5. **インデント（字下げ）が綺麗か？**
   - コードが読みやすく整形されているか

**解答例**：[01-03.html](solutions/01-03.html)

## 応用編

### 演習 4：エラーを見つけよう

以下の HTML には問題があります。何が間違っているか見つけて修正してください：

```html
<html>
  <head>
    <title>テストページ</title>
  </head>
  <body>
    <h1>こんにちは</h1>
    <p>これはテストです</p>
  </body>
</html>
```

**ヒント**：少なくとも 3 つの問題があるよ！

<details markdown="1">
<summary>答えを見る</summary>

**問題点**：

1. `<!DOCTYPE html>`がない
2. `<html>`に`lang`属性がない
3. `<meta charset="UTF-8">`がない

**修正版**：

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>テストページ</title>
  </head>
  <body>
    <h1>こんにちは</h1>
    <p>これはテストです</p>
  </body>
</html>
```

</details>
