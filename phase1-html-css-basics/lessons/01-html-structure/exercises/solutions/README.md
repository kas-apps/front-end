### exercise01.html - 演習 1 の解答

**演習内容**：最初の HTML ファイルを作ろう

- タイトル：「自己紹介」
- 言語：日本語
- body 内容：
  - 見出し（h1）：名前
  - 段落（p）：簡単な自己紹介文

**ポイント**：

- ✅ HTML5 の基本構造を使用

- ✅ コメントで各部分の役割を説明

### exercise02.html - 演習 2 の解答

**追加した要素**：

```html
<meta name="description" content="山田太郎の自己紹介ページです" />
```

**ポイント**：

- ✅ meta description を追加
- ✅ 検索エンジンに表示される説明文を設定
- ✅ SEO 対策の基本を実践

### exercise03.html - 演習 3 の解答

**演習内容**：AI に指示して作ってもらおう

- タイトル：「私の好きな食べ物」
- body 内容：
  - h1 見出し：「好きな食べ物トップ 3」
  - 3 つの段落、それぞれ好きな食べ物について

**ポイント**：

- ✅ 完全な HTML5 構造
- ✅ meta description を含む
- ✅ 3 つの段落で好きな食べ物を説明
- ✅ コメントで各段落の意味を明示

**AI への指示例**：

```text
「HTML5の基本構造で、タイトルが『私の好きな食べ物』のHTMLファイルを作ってください。
bodyには、『好きな食べ物トップ3』という見出し（h1）と、
3つの段落（p）で好きな食べ物について説明してください。
文字コード、viewport、meta descriptionも含めてください。」
```

---

## 🎯 学習のポイント

これらの解答例を見る時は、以下のポイントに注目しよう：

1. **HTML の基本構造**

   - DOCTYPE、html、head、body が揃っている
   - 正しくネストされている

2. **必須要素**

   - `<meta charset="UTF-8">` がある
   - `<title>` が適切に設定されている
   - `lang="ja"` で日本語を指定している

3. **コメント**

   - 各部分の役割が分かりやすく説明されている
   - 実務でも同じようにコメントを書くと、後で見返しやすい

4. **インデント**
   - ネストが視覚的に分かりやすい
   - 読みやすいコードは、メンテナンスしやすい
