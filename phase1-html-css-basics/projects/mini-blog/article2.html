CSSでデザインしよう - スタイリングの基本

2025年1月10日 投稿

CSSとは？

CSS（Cascading Style Sheets）は、HTMLで作った構造に「見た目」を与える言語です。 色、フォント、レイアウト、余白など、Webページのデザインを自由にコントロールできます。

CSSの書き方

CSSは「セレクタ」「プロパティ」「値」の3つの要素で構成されます：

  • セレクタ：どの要素にスタイルを適用するか（例：h1, .class, #id）
  • プロパティ：何を変更するか（例：color, font-size）
  • 値：どう変更するか（例：blue, 16px）

基本的なCSSプロパティ

1. 色（color, background-color）

文字色や背景色を指定します。色名（blue）、16進数（#3b82f6）、RGB（rgb(59, 130, 246)）などで指定できます。

2. フォント（font-size, font-family, font-weight）

文字のサイズ、書体、太さを変更します。font-sizeはpxやremで指定します。

3. ボックスモデル（margin, padding, border）

要素の余白や枠線を設定します。marginは外側の余白、paddingは内側の余白です。

4. テキストの配置（text-align, line-height）

テキストの配置（左寄せ、中央寄せ、右寄せ）や行間を調整して、読みやすくします。

CSSの適用方法

CSSをHTMLに適用する方法は3つあります：

  • インラインCSS：HTMLタグのstyle属性に直接書く（あまり推奨されない）
  • 内部CSS：<head>内の<style>タグに書く
  • 外部CSS：別ファイル（.css）に書いて、<link>で読み込む（推奨！）

まとめ

CSSを使えば、HTMLの骨組みを美しくデザインできます。色、フォント、余白を調整して、 あなただけのオリジナルなWebページを作りましょう！
