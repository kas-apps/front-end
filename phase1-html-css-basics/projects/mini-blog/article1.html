HTMLの基礎を学ぼう - 初心者のためのガイド

2025年1月15日 投稿

HTMLとは？

HTML（HyperText Markup Language）は、Webページの構造を定義するためのマークアップ言語です。テキストや画像、リンクなどをタグで囲むことで、ブラウザに「これは見出し」「これは段落」といった情報を伝えます。

HTMLの基本構造

すべてのHTMLドキュメントは、以下の基本構造を持っています：

  • <!DOCTYPE html>：HTML5であることを宣言
  • <html>：HTMLドキュメント全体を囲む
  • <head>：メタ情報（タイトル、文字コードなど）
  • <body>：ページに表示される実際のコンテンツ

重要なHTML要素

1. 見出し（h1-h6）

見出しは文書の構造を表します。h1が最も重要で、h6が最も小さな見出しです。

2. 段落（p）

テキストの段落を表します。文章を読みやすく構造化するために使います。

3. リスト（ul, ol, li）

情報を箇条書きで表示します。ulは順序なしリスト、olは順序付きリストです。

4. リンク（a）

他のページやセクションへのリンクを作成します。href属性にURLを指定します。

まとめ

HTMLはWebページの基礎です。タグの意味を理解して、適切に使い分けることで、 構造的で読みやすいWebページを作ることができます。次は、CSSでページをスタイリングする方法を学びましょう！
