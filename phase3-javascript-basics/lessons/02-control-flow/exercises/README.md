## 基礎編

### 問題 2-1: if 文で成人判定

以下の要件を満たす JavaScript コードを書いてください：

1. 変数 `age` に年齢（数値）を代入する
2. if 文を使って、18 歳以上なら「成人です」、そうでなければ「未成年です」と出力する

**期待される出力例（age = 20 の場合）**：

```
成人です
```

---

### 問題 2-2: else if で成績判定

以下の要件を満たす JavaScript コードを書いてください：

1. 変数 `score` に点数（0-100 の数値）を代入する
2. if-else if-else を使って、以下のように成績を判定する：
   - 90 点以上：「A」
   - 80 点以上 90 点未満：「B」
   - 70 点以上 80 点未満：「C」
   - 60 点以上 70 点未満：「D」
   - 60 点未満：「F」
3. 「あなたの成績は〇です」と出力する

**期待される出力例（score = 85 の場合）**：

```
あなたの成績は B です
```

---

### 問題 2-3: for ループで数字を表示

以下の要件を満たす JavaScript コードを書いてください：

1. for ループを使って、1 から 10 までの数字を表示する
2. 各数字を「数字: 〇」という形式で出力する

**期待される出力**：

```
数字: 1
数字: 2
数字: 3
...
数字: 10
```

## 応用編

### 問題 2-4: 偶数だけ表示

以下の要件を満たす JavaScript コードを書いてください：

1. for ループを使って、1 から 20 までの数字をチェックする
2. if 文を使って、偶数の時だけ表示する
3. 偶数の個数も数えて、最後に「偶数は全部で〇個ありました」と出力する

**期待される出力**：

```
2
4
6
...
20
偶数は全部で 10 個ありました
```

### 問題 2-5: 合計を計算

以下の要件を満たす JavaScript コードを書いてください：

1. for ループを使って、1 から 100 までの数字を足し合わせる
2. 途中経過（10, 20, 30...ごと）も表示する
3. 最終的な合計を出力する

**期待される出力**：

```
10までの合計: 55
20までの合計: 210
30までの合計: 465
...
100までの合計: 5050
```

**ヒント**：`i % 10 === 0` で 10 の倍数かどうかチェックできるよ

### 問題 2-6: switch 文で曜日メッセージ

以下の要件を満たす JavaScript コードを書いてください：

1. 変数 `dayNumber` に曜日の番号（1-7）を代入する
   - 1: 月曜日、2: 火曜日、...、7: 日曜日
2. switch 文を使って、曜日に応じたメッセージを表示する
   - 月曜日: 「今週も頑張ろう！」
   - 火曜日〜木曜日: 「まだまだ続く...」
   - 金曜日: 「明日は休み！」
   - 土曜日・日曜日: 「週末だ！」
   - それ以外: 「無効な曜日番号です」

**期待される出力例（dayNumber = 5 の場合）**：

```
金曜日: 明日は休み！
```

## チャレンジ編

### 問題 2-7: 九九の表を作る

以下の要件を満たす JavaScript コードを書いてください：

1. 二重の for ループを使って、九九の表を作成する
2. 各行の計算結果を見やすく出力する
3. 各段（1 の段、2 の段...）の間に区切り線を入れる

**期待される出力例**：

```
【 1 の段 】
1 × 1 = 1
1 × 2 = 2
...
1 × 9 = 9
-----------
【 2 の段 】
2 × 1 = 2
2 × 2 = 4
...
2 × 9 = 18
-----------
...
【 9 の段 】
9 × 1 = 9
9 × 2 = 18
...
9 × 9 = 81
```

**ヒント**：

- 外側のループ（`i`）は段数（1-9）
- 内側のループ（`j`）は掛ける数（1-9）
- `console.log()` で見やすく整形しよう
