## 基礎編

### 問題 1-1: 変数の宣言と出力

以下の要件を満たす JavaScript コードを書いてください：

1. 自分の名前を格納する変数 `myName` を `const` で宣言する
2. 自分の年齢を格納する変数 `myAge` を `const` で宣言する
3. 自分の出身地を格納する変数 `myHometown` を `const` で宣言する
4. これらの変数の値を `console.log()` で出力する

**期待される出力例**：

```
太郎
25
東京
```

---

### 問題 1-2: データ型の確認

以下の要件を満たす JavaScript コードを書いてください：

1. 数値 `100` を格納する変数 `num` を宣言する
2. 文字列 `"100"` を格納する変数 `str` を宣言する
3. 真偽値 `true` を格納する変数 `bool` を宣言する
4. `null` を格納する変数 `emptyValue` を宣言する
5. 値を代入しない変数 `notAssigned` を宣言する
6. それぞれの変数の**値**と**型**を `console.log()` で出力する

**期待される出力例**：

```
値: 100, 型: number
値: 100, 型: string
値: true, 型: boolean
値: null, 型: object
値: undefined, 型: undefined
```

---

### 問題 1-3: 基本的な計算

以下の要件を満たす JavaScript コードを書いてください：

1. 商品の価格 `800` を `price` に代入する
2. 購入数量 `5` を `quantity` に代入する
3. 小計（`price × quantity`）を計算して `subtotal` に代入する
4. 消費税率 `0.1`（10%）を `taxRate` に代入する
5. 消費税額（`subtotal × taxRate`）を計算して `tax` に代入する
6. 合計金額（`subtotal + tax`）を計算して `total` に代入する
7. それぞれの値を分かりやすく `console.log()` で出力する

**期待される出力例**：

```
商品価格: 800円
購入数量: 5個
小計: 4000円
消費税: 400円
合計金額: 4400円
```

## 応用編

### 問題 1-4: テンプレートリテラルで自己紹介

以下の要件を満たす JavaScript コードを書いてください：

1. 名前、年齢、趣味を変数に代入する
2. テンプレートリテラルを使って、以下の形式の自己紹介文を作成する
3. 作成した文章を `console.log()` で出力する

**自己紹介文の形式**：

```
こんにちは！私の名前は〇〇です。
年齢は△△歳で、趣味は××です。
よろしくお願いします！
```

### 問題 1-5: 比較演算子の活用

以下の要件を満たす JavaScript コードを書いてください：

1. 変数 `score` に点数（0-100 の数値）を代入する
2. `score` が 80 以上かどうかを判定し、結果を `isHighScore` に代入する
3. `score` が 60 未満かどうかを判定し、結果を `isLowScore` に代入する
4. `score` がちょうど 100 かどうかを判定し、結果を `isPerfect` に代入する
5. それぞれの判定結果を分かりやすく出力する

**期待される出力例（score = 85 の場合）**：

```
点数: 85点
80点以上: true
60点未満: false
満点: false
```

---

### 問題 1-6: 複数の演算子を組み合わせる

以下の要件を満たす JavaScript コードを書いてください：

1. 変数 `a` に `10` を、`b` に `20` を代入する
2. `a` と `b` の和、差、積、商、余りを計算する
3. `a` が `b` より小さいかどうかを判定する
4. `a` と `b` の両方が 10 以上かどうかを判定する
5. すべての結果をテンプレートリテラルを使って分かりやすく出力する

**期待される出力例**：

```
a = 10, b = 20
和: 30
差: -10
積: 200
商: 0.5
余り: 10
aはbより小さい: true
両方とも10以上: true
```

## チャレンジ編

### 問題 1-7: 買い物の明細書を作る

以下の要件を満たす JavaScript コードを書いてください：

1. 3 つの商品（商品名と価格）を変数に代入する
2. 各商品の購入数量を変数に代入する
3. 各商品の小計を計算する
4. すべての商品の合計金額を計算する
5. 消費税（10%）を計算する
6. 最終的な支払い金額を計算する
7. 買い物明細を分かりやすく出力する

**期待される出力例**：

```
=== 買い物明細 ===
りんご × 3個: 300円
バナナ × 2個: 200円
オレンジ × 5個: 500円
---
小計: 1000円
消費税(10%): 100円
---
合計金額: 1100円
```

**ヒント**：

- テンプレートリテラルを活用しよう
- 計算を段階的に行って、途中結果も変数に入れよう
- 出力を見やすくするために、改行や記号を使おう
