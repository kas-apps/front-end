# Lesson 5: 基本的なDOM操作 - 演習問題 📝

## 基礎編

### 問題 5-1: 要素の取得と内容変更
ID が "message" の段落要素を取得し、テキストを「Hello, DOM!」に変更してください。

### 問題 5-2: スタイルの変更
ID が "box" の div 要素を取得し、背景色を「lightblue」、高さを「100px」に変更してください。

### 問題 5-3: クリックイベント
ボタンをクリックすると、段落のテキストが「クリックされました！」に変わるコードを書いてください。

## 応用編

### 問題 5-4: 入力のリアルタイム表示
入力フィールドに入力した内容を、リアルタイムで別の要素に表示してください。

### 問題 5-5: クラスのトグル
ボタンをクリックすると、要素に "highlight" クラスがトグル（追加/削除）されるコードを書いてください。

### 問題 5-6: 要素の追加
ボタンをクリックすると、リストに新しい項目が追加されるコードを書いてください。

## チャレンジ編

### 問題 5-7: シンプルなカウンター
+1, -1, リセットボタンを持つカウンターアプリを作成してください。
