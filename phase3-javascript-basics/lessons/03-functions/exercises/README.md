## 基礎編

### 問題 3-1: 簡単な関数を作る

以下の要件を満たす JavaScript コードを書いてください：

1. `sayHello` という名前の関数を作成する
2. 関数は「Hello, World!」と console.log で出力する
3. 関数を 3 回呼び出す

**期待される出力**：

```
Hello, World!
Hello, World!
Hello, World!
```

---

### 問題 3-2: 引数を使った関数

以下の要件を満たす JavaScript コードを書いてください：

1. `greet` という名前の関数を作成する
2. 関数は `name` という引数を 1 つ受け取る
3. 「こんにちは、〇〇さん！」と出力する
4. 「太郎」「花子」「次郎」の 3 人で関数を呼び出す

---

### 問題 3-3: 戻り値を返す関数

以下の要件を満たす JavaScript コードを書いてください：

1. `add` という名前の関数を作成する
2. 関数は 2 つの数値（`a`, `b`）を引数に取る
3. 2 つの数値の合計を return で返す
4. `add(5, 3)`, `add(10, 20)`, `add(100, 200)` を呼び出して結果を出力する

## 応用編

### 問題 3-4: アロー関数で書き直す

以下の関数をアロー関数で書き直してください：

```javascript
function multiply(a, b) {
  return a * b;
}
```

書き直した後、`multiply(4, 5)` を呼び出して結果を出力してください。

---

### 問題 3-5: デフォルト引数を使う

以下の要件を満たす JavaScript コードを書いてください：

1. `calculatePrice` という名前の関数を作成する
2. 引数：`price`（価格）、`taxRate`（税率、デフォルト値 0.1）
3. 税込み価格を計算して返す
4. 以下のパターンで呼び出して結果を出力：
   - `calculatePrice(1000)` - デフォルトの税率
   - `calculatePrice(1000, 0.08)` - 税率 8%
   - `calculatePrice(5000)` - デフォルトの税率

---

### 問題 3-6: 複数の関数を組み合わせる

以下の要件を満たす JavaScript コードを書いてください：

1. `double` 関数：数値を 2 倍にして返す
2. `square` 関数：数値を 2 乗して返す
3. 数値 5 を、まず `double` して、その結果を `square` する
4. 最終結果を出力する（5 → 10 → 100）

---

## チャレンジ編

### 問題 3-7: FizzBuzz 関数を作る

以下の要件を満たす JavaScript コードを書いてください：

1. `fizzBuzz` という名前の関数を作成する
2. 引数：`num`（数値）
3. ルール：
   - 3 の倍数なら「Fizz」を返す
   - 5 の倍数なら「Buzz」を返す
   - 3 と 5 両方の倍数なら「FizzBuzz」を返す
   - それ以外はそのまま数値を返す
4. for ループで 1 から 30 まで繰り返し、各数値で `fizzBuzz` を呼び出して結果を出力する

**期待される出力例**：

```
1
2
Fizz
4
Buzz
Fizz
7
...
FizzBuzz
```
