## HTML/CSS はプログラミング言語？

厳密には異なるが、**HTML/CSS こそが Web の土台**

| 特徴   | JavaScript                 | HTML/CSS                                                    |
| ------ | -------------------------- | ----------------------------------------------------------- |
| 目的   | ロジックや処理を記述       | 構造とスタイルを記述                                        |
| 実行   | 順次実行、条件分岐、ループ | 宣言的に表示を指定                                          |
| 呼び方 | プログラミング言語         | **マークアップ言語** (HTML)<br>**スタイルシート言語** (CSS) |

## HTML と CSS の役割

<div style="display: flex; gap: 24px;">
<div>

### HTML： ドキュメントの**構造**を定義

「何が」「どんな意味で」あるかを定義する

- 見出し、段落、ボタン...

### CSS： ドキュメントを**装飾**

「どう見えるか」を定義する

- 色、サイズ、配置、余白...

</div>

<div style="flex: auto; font-size: 24px;">

`html`

```html
<article>
  <h1>バイブコーディング入門</h1>
  <p>AI と一緒にコードを書くスタイル</p>
  <button>詳しく見る</button>
</article>
```

`css`

```css
h1 {
  color: blue;
  font-size: 32px;
  font-weight: bold;
}
```

</div>

</div>

## あなたはもうドキュメント作成を知っている

Word や Pages で文書を作ることと、HTML/CSS は**本質的に同じ**

| Word, Pages          | HTML/CSS                |
| -------------------- | ----------------------- |
| 見出し 1 を設定      | `<h1>タイトル</h1>`     |
| 見出し 2 を設定      | `<h2>サブタイトル</h2>` |
| 本文を書く           | `<p>本文</p>`           |
| 太字にする           | `<strong>重要</strong>` |
| フォントや色を変える | CSS: `color: blue;`     |

## Phase 0 で押さえておくべきポイント

- **AI 時代だからこそ基礎が重要**であることを理解した
- **HTML/CSS は Web の土台・必須知識**であることを認識した
- **HTML = 構造、CSS = 装飾**の役割分担を理解した
- **HTML/CSS は Word/Pages と本質的に同じ**ことが分かった
- AI との協働がスムーズになることに期待が高まった
