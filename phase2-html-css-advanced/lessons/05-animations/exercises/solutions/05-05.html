演習 5：ローディングスピナー

ポイント

円の一部だけ色を付けて、それを回転させることで、
ローディング中を表現するスピナーが作れます。

重要な設定

  • border-radius: 50% - 正方形を円形に
  • linear - 一定速度で回転
  • infinite - 無限ループ
  • rotate(360deg) - 1回転
CSS のポイント：
@keyframes spin {
  0% { transform: rotate(0deg); }
  100% { transform: rotate(360deg); }
}

.spinner {
  border: 4px solid #f3f3f3;
  border-top: 4px solid #3498db;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}
