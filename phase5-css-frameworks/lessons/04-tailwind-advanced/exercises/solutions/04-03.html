アニメーションボタンコレクション

Tailwind CSSのユーティリティクラスを使った
インタラクティブなボタンアニメーション

Scale Button

ホバー時に拡大するシンプルなエフェクト

hover:scale-110

Float Button

上に浮き上がるような動き

hover:-translate-y-2 hover:shadow-xl

Rotate Button

軽く回転する遊び心のあるエフェクト

hover:rotate-3

Combined Button

複数のエフェクトを組み合わせた豪華な動き

hover:scale-105 hover:-translate-y-1 hover:shadow-2xl

ボーナス: パルスエフェクト

注目を集める継続的なアニメーション

animate-ping + scale + shadow

ポイント: すべてのアニメーションに transition と duration-300 を適用することで、 スムーズなアニメーションを実現しています。
