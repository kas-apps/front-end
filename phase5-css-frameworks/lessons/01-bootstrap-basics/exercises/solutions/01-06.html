✓ ポイント

  • d-flex: Flexboxを有効化（display: flex）
  • justify-content-center: 横方向中央寄せ
  • justify-content-between: 両端寄せ（要素間の間隔を均等に）
  • justify-content-around: 周囲均等（要素の周囲の間隔を均等に）
  • justify-content-start / end: 左寄せ / 右寄せ
  • align-items-center: 縦方向中央寄せ（親要素に高さが必要）
  • flex-column: 縦並び（デフォルトは横並び）
  • m-2: マージン（margin: 0.5rem）
  • p-3: パディング（padding: 1rem）
