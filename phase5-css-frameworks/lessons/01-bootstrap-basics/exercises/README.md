## 基礎編（3問）

### 演習 1-1: 2 カラムレイアウト

**難易度**：⭐

**課題**：
Bootstrap のグリッドシステムを使って、以下の仕様を満たす 2 カラムレイアウトを作成してください。

**仕様**：
- コンテナを使用
- 左カラム：8 カラム分
- 右カラム：4 カラム分
- 各カラムに背景色とパディングを設定
- 左カラムには「メインコンテンツ」、右カラムには「サイドバー」と表示

**ヒント**：
- `container`、`row`、`col-*` を使う
- 背景色は `bg-light`、`bg-info` など
- パディングは `p-3` などを使う

---

### 演習 1-2: 色とりどりのボタン

**難易度**：⭐

**課題**：
Bootstrap のボタンコンポーネントを使って、様々なバリエーションのボタンを作成してください。

**仕様**：
- Primary、Success、Danger、Warning の 4 種類のボタン
- それぞれのアウトラインバージョンも作成
- 大きいサイズ、通常サイズ、小さいサイズのボタンを作成
- 全てのボタンに適切なテキストを設定

**ヒント**：
- `btn btn-*` でカラーを指定
- `btn-outline-*` でアウトライン版
- `btn-lg`、`btn-sm` でサイズ変更

---

### 演習 1-3: カードコンポーネント

**難易度**：⭐⭐

**課題**：
Bootstrap のカードコンポーネントを使って、商品カードを作成してください。

**仕様**：
- カード内に画像を表示（https://via.placeholder.com/400x200 を使用）
- カードタイトル：「商品名」
- カード本文：「この商品の説明文がここに入ります」
- ボタン：「カートに追加」（Primary カラー）
- カードの幅は最大 400px

**ヒント**：
- `card`、`card-img-top`、`card-body` を使う
- `card-title`、`card-text` でテキスト
- `style="max-width: 400px"` で幅を制限

## 応用編（3問）

### 演習 1-4: レスポンシブ 3 カラムレイアウト

**難易度**：⭐⭐

**課題**：
レスポンシブな 3 カラムレイアウトを作成してください。

**仕様**：
- スマホ（< 768px）：1 列（縦並び）
- タブレット（≥ 768px）：2 列
- PC（≥ 992px）：3 列
- 6 つのカードを配置
- 各カードには番号（1-6）と簡単な説明文を表示
- カードの高さを揃える

**ヒント**：
- `col-12 col-md-6 col-lg-4` のような組み合わせ
- `h-100` でカードの高さを揃える
- `g-4` で余白を設定

---

### 演習 1-5: アラートメッセージシステム

**難易度**：⭐⭐

**課題**：
様々な種類のアラートメッセージを表示するセクションを作成してください。

**仕様**：
- Success アラート：「✓ 登録が完了しました！」
- Info アラート：「ℹ️ 新機能が追加されました。詳細を確認してください。」
- Warning アラート：「⚠️ パスワードの有効期限が近づいています。」
- Danger アラート：「❌ エラーが発生しました。もう一度お試しください。」
- 1 つのアラートには見出し（alert-heading）と区切り線（hr）を含める
- 1 つのアラートには閉じるボタンを含める

**ヒント**：
- `alert alert-*` で種類を指定
- `alert-heading` で見出し
- `alert-dismissible fade show` と `btn-close` で閉じるボタン
- JavaScript の読み込みが必要

---

### 演習 1-6: フレキシブルレイアウト 🤖

**難易度**：⭐⭐⭐

**課題**：
Bootstrap の Flexbox ユーティリティを使って、要素を配置してください。

**仕様**：
1. 3 つのボックスを横並びで中央配置
2. 3 つのボックスを両端寄せ（左、中央、右）
3. 3 つのボックスを縦方向にも中央配置（高さ 300px のコンテナ内）
4. 各ボックスには背景色とパディングを設定

**ヒント**：
- `d-flex` で Flexbox 有効化
- `justify-content-center`、`justify-content-between` で横方向配置
- `align-items-center` で縦方向配置
- `bg-primary`、`text-white`、`p-3` などで装飾

**AI への指示例**（参考）：
```
Bootstrap 5のFlexboxユーティリティクラスを使って、
以下の3つのレイアウトを作成してください：

1. 3つのボックスを横並びで中央配置
2. 3つのボックスを両端寄せ
3. 3つのボックスを縦横中央配置（コンテナ高さ300px）

各ボックスには背景色とパディングを設定してください。
```

## チャレンジ編（1問）

### 演習 1-7: シンプルなランディングページ

**難易度**：⭐⭐⭐⭐

**課題**：
これまで学んだことを全て使って、シンプルなランディングページを作成してください。

**仕様**：

**1. ヒーローセクション**
- 背景色：Primary
- 白いテキスト
- 大きな見出し（display-4）
- リード文（lead）
- CTA ボタン（大きいサイズ、白背景）

**2. 特徴セクション**
- 3 カラムのカードレイアウト
- レスポンシブ対応（スマホ:1 列、PC:3 列）
- 各カードには絵文字、見出し、説明文
- カードの高さを揃える

**3. お客様の声セクション**
- 2 カラムレイアウト（スマホでは 1 列）
- 各カードにはユーザーアイコン（絵文字でOK）、名前、コメント

**4. CTA セクション**
- 背景色：Light
- 中央寄せのテキストとボタン

**5. フッター**
- 背景色：Dark
- 白いテキスト
- コピーライト表記
- 中央寄せ

**ヒント**：
- セクションごとに `container` を使う
- セクション間の余白は `my-5` などで調整
- 実際のランディングページを参考にする
- まず全体の構造を作ってから、細部を調整

**AI への指示例**（参考）：
```
Bootstrap 5を使って、以下の構成のランディングページを作成してください：

【ヒーローセクション】
- 背景色Primary、白いテキスト
- 大見出し（display-4）とリード文（lead）
- 大きなCTAボタン

【特徴セクション（3カラム）】
- レスポンシブ対応（スマホ:1列、PC:3列）
- 各カードに絵文字、見出し、説明文
- カードの高さを揃える

【お客様の声（2カラム）】
- レスポンシブ対応
- ユーザー名とコメント

【CTAセクション】
- 背景色Light
- 中央寄せのボタン

【フッター】
- 背景色Dark、白いテキスト
- コピーライト表記

CDN、viewport、適切なスペーシングを含めてください。
```
